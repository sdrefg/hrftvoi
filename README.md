### 变量

对部署时需设定的变量名称做如下说明。

| 变量 | 默认值 | 说明 |
| :--- | :--- | :--- |
| `ID` | `` | VLESS 用户 ID，用于身份验证，为 UUID 格式 |
| `WSPATH` | `/app` | WebSocket 所使用的 HTTP 协议路径 |
